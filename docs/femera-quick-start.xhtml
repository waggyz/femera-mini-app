Femera 0.3 quick start guide

David Wagner
Abstract
Femera is a high-performance computing mini-app for testing and tuning numerical algorithms used to solve partial differential equations over a discrete space-time domain. Femera is not a full-featured application as it implements only the basic capabilities of a complete modeling, analysis, and visualization simulation framework. This quick start guide summarizes how to build and install Femera in its default configuration and introduces several ways to begin using the mini-app. Readers should have some knowledge of how to build Linux applications from source.

1 About this quick start guide

Femera is not distributed with a ready-to-use binary executable as it takes advantage of features of the specific hardware it is used on to perform well in a high-performance parallel computing environment. This quick start guide is intended for users to build, install, and begin using the recommended Femera configuration.

System administrators: build and install Femera for use by production and power users.

Production users: run Femera from the command line (section subsec:cli-examples), use Femera interactively (section subsec:Intro-python-script), or write Femera Python scripts (section subsec:Intro-python-interactive).

Power users: develop Femera plugins and compile them in a compatible build environment.

Femera developers: maintain, debug, extend, and customize Femera.

System administrators, power users, and developers should review the Femera user and developer guide before proceeding with this quick start. Femera is only available for Linux and requires a build environment compatible with the intended run-time environment.

2 Linux system requirements

A minimal Femera configuration can be built and run in most modern Linux environments with at least one of the compiler toolchains listed in table 1. The compiler toolchain must support C++11 and should support Open Multi-processing (OpenMP). In addition, the software listed in table 1 should be available, on the build system and in the run-time environment, to build and run the recommended Femera configuration.
