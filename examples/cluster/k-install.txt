NASA LaRC K-cluster Femera installation instructions

NOTE Everything builds much faster if you clone Femera in hpnobackup1.

Follow the installation instructions in the README.md located in the Femera root
directory, with the following change to step 3.

Step 3.

* Load the modules listed in k-modules.sh. You may replace the anaconda and
  Python modules with your preferred Python 3 environment.

* Copy tools/cluster/config.k to config.local in the Femera root directory.

Continute with the installation instructions.
